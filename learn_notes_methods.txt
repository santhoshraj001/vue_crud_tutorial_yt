13.retrieving events from the Child
 -> receives data and events form child 
 eg: <Employee-Forms @employee-updated="receiveEmployee" @employee-details="receiveDetails"/> // @employee-updated this is event and emit name from child
  methods: {
     receiveEmployee(data) {
    console.log("From Child:", data)
  },
  receiveDetails(employee) {
    const newEmployee = { ...employee, id: this.employees.length + 1 }  // this is new object and this is add id property in form data and this is one way data flow from child to parent component
    this.employees = [...this.employees, newEmployee]  //(...this.employees means old employees array) this is push form data to employees array and this is one way data flow from child to parent component
   //  this.employees.push(employee)   //this is another same method to add value in employees array
  },

}

--------------------pass data from parent to child-----------------
eg:
app.vue-----(parent)
<Employee-Table :employees="employees"/>
What happens?
: means v-bind
Parent passes its employees data
Child component receives it

<script>
export default {
  data() {
    return {
      employees: [
        {id: 1, name: 'John Doe', email: 'john@example.com', 
        office: 'Covai', age: 25, salary: 55000 },   //this is reactive data
        {id: 2, name: 'Jane Smith', email: 'jane@example.com', 
        office: 'Chennai', age: 30, salary: 60000 }, // this name is one property and all fields name is call like a property
      ],
    }
  },
}
</script>

employee_table.vue----(child)
----this is child component of employee_table.vue------
<tbody>
                <tr v-for="employee in employees" :key="employee.id">
                    <td>{{ employee.name }} test</td>
                    <td>{{ employee.email }}</td>
                    <td>{{ employee.office }}</td>
                    <td>{{ employee.age }}</td>
                    <td>{{ employee.salary }}</td>
                    
                </tr>
            </tbody>
        </table>
    </div>
</template>
<script>
export default {
    name: 'Employee-Table' ,  // this is component name use in app.vue inside template tag
    props: {
        employees: Array, // this data type is array and this is to receive data from parent component (app.vue)
    }  
}

</script>

Reactive data = data that automatically updates the view when its value changes.
